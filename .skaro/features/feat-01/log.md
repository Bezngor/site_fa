**User:** Название: Лендинг FactoryAll — полная разработка

Описание:
Разработать одностраничный лендинг консалтингового агентства FactoryAll 
на Next.js 15 (SSG, App Router, Tailwind CSS 4, TypeScript).

Цель: конвертировать посетителей в заявки через форму «Записаться 
на бесплатную диагностику».

Структура (8 секций):
1. HeroSection — заголовок, подзаголовок, CTA-кнопка
2. ProblemsSection — 4 боли целевой аудитории
3. HowWeWorkSection — 5 этапов работы
4. CasesSection — 3 кейса из реальных проектов
5. ProductLineSection — 3 пакета услуг
6. ResultsSection — измеримые результаты (цифры)
7. AboutSection — 3 отличия FactoryAll
8. CtaFormSection — форма + Formspree + fallback mailto

Технические требования:
- Все тексты и данные в /lib/data/ (cases.ts, texts.ts, products.ts)
- Server Components по умолчанию, 'use client' только для CtaFormSection
- Zod-валидация формы на клиенте
- Цветовая палитра: #1B2A4A (navy), #F59E0B (amber), #FFFFFF
- Логотип: текстовый «F!» в navbar
- Адаптивная вёрстка (mobile-first)
- Деплой: Vercel

Критерии готовности:
- Все 8 секций отрендерены
- Форма отправляет данные на Formspree, fallback на mailto при ошибке
- Lighthouse score ≥ 90 (Performance, SEO)
- Bundle size < 150 KB gzipped

## 2026-03-21 20:50

**User:** Ответы на вопросы:
1. Навигация: Sticky Header с якорными ссылками на все 8 секций + 
   логотип «F!» слева. На мобильных — бургер-меню (hamburger).
2. Кейсы: Карточки с inline expand (CSS accordion, без модалов). 
   На карточке: заголовок + проблема + результат. 
   По клику — раскрывается полное описание.
3. Изображения: Готовых ассетов нет. SVG-иконки для этапов и 
   преимуществ. Для кейсов — solid color плашки с иконкой.
4. Formspree: Конфигурировать через NEXT_PUBLIC_FORMSPREE_ID в .env. 
   На старте — placeholder значение.
