# AI_NOTES — Этап 3: Реализация интерактивной CasesSection

## Что было сделано
- Реализован компонент `CasesSection` с использованием `'use client'`.
- Добавлено состояние `useState` для независимого управления раскрытием 3-х кейсов.
- Реализована анимация раскрытия через CSS Grid (`grid-template-rows: 0fr` -> `1fr`) и `transition` в `app/globals.css`.
- Использована иконка `ChevronDown` из `lucide-react` с точечным импортом.
- Контент кейсов динамически подгружается из `lib/data/cases.ts`.

## Почему выбран такой подход
- Использование `grid-template-rows` позволяет плавно анимировать высоту блока с неизвестным контентом без использования JavaScript-библиотек (Framer Motion), что экономит размер бандла.
- Независимое состояние массива `boolean` обеспечивает гибкость взаимодействия для пользователя.
