## Соответствие инвариантам архитектуры
- [x] Статическая генерация (SSG) — компоненты являются клиентскими только при необходимости, но данные UI-компоненты могут быть Server Components (они не используют state). В текущей реализации они не используют 'use client', что допустимо.
- [x] Bundle size <150 KB — компоненты легковесны, используют только Tailwind CSS.
- [x] Безопасность — honeypot будет добавлен позже.
- [x] Язык — все строки на русском (в текущих компонентах нет строк, только структура; тексты будут передаваться через props из texts.ts).

## Как проверить
1. Убедиться, что все файлы из списка существуют в проекте.
2. Проверить, что Button принимает пропс `isLoading` и отображает спиннер.
3. Проверить, что FormInput и FormTextarea отображают ошибку при передаче пропса `error`.
4. Проверить, что ErrorMessage и SuccessMessage рендерятся с соответствующим стилем и контентом.
5. Запустить ESLint и TypeScript компиляцию для проверки типов.
