# AI_NOTES — Этап 2: Сборка главной страницы, Header и стили для навигации

## Что было сделано
- Обновлен `app/globals.css`: добавлено свойство `scroll-behavior: smooth` для обеспечения плавной прокрутки.
- Обновлен `app/page.tsx`: реализован последовательный рендер всех 8 секций, импортированных из `components/sections`.
- Обновлен `components/layout/Header.tsx`: реализована sticky-навигация с использованием `backdrop-blur` и ссылками на все 8 ID секций.

## Почему выбран такой подход
- Использование `sticky top-0` с `backdrop-blur` обеспечивает современный вид навигации, которая всегда остается доступной пользователю.
- Плавная прокрутка через CSS является наиболее производительным и легким решением, соответствующим требованиям NFR-01.
