# Спецификация: hero-section

## Контекст
Hero — первая секция для привлечения внимания, с акцентом на primary цвета (ADR-002). Она формирует первое впечатление у пользователя и направляет его к целевому действию.

## User Scenarios
1. **Посетитель:** Заходит на главную страницу, видит ключевое предложение (H1), нажимает на кнопку CTA, после чего происходит плавный скролл к форме обратной связи.

## Functional Requirements
- **FR-01:** Импорт текстового контента из `lib/data/texts.ts`.
- **FR-02:** Рендер заголовка H1, состоящего из основного текста и подзаголовка (span) для SEO-оптимизации.
- **FR-03:** Использование фонового изображения через `next/image` (формат WebP). Путь к файлу: `/images/hero-bg.webp`.
- **FR-04:** Наложение градиентного слоя (overlay) поверх изображения (от темного #1B2A4A к прозрачному) для обеспечения читаемости текста.
- **FR-05:** Реализация кнопки CTA как якорной ссылки (`anchor link`) на секцию `#cta-form`.
- **FR-06:** Секция должна занимать всю высоту экрана (`min-h-screen`).

## Non-Functional Requirements
- **NFR-01:** Адаптивность (mobile-first дизайн).
- **NFR-02:** Плавная прокрутка (smooth scroll) средствами CSS.
- **NFR-03:** Высокая производительность (LCP) за счет приоритетной загрузки изображения (`priority` атрибут в `next/image`).

## Boundaries (what is NOT included)
- Глобальный компонент Button (на данном этапе создать локальный UI-компонент или использовать стилизованный тег `<a>`).
- Сложные анимации появления элементов (только базовый CSS).

## Acceptance Criteria
- [x] Секция корректно рендерится в `app/page.tsx`.
- [x] Фоновое изображение оптимизировано, используется WebP и атрибут `priority`.
- [x] Заголовок H1 содержит ключевые слова и разделен на логические части согласно A4.
- [x] Кнопка CTA плавно перемещает пользователя к форме (проверено наличие `scroll-behavior: smooth`).
- [x] Текст остается читаемым на любых разрешениях благодаря градиентному оверлею.
- [x] Секция занимает 100% высоты вьюпорта на десктопах и мобильных устройствах.
