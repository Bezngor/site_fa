# AI_NOTES — Реализация ProblemsSection

## Что было сделано
- Обновлен файл `lib/data/texts.ts`: добавлена секция `problems` с заголовком, описанием и списком строк.
- Обновлен файл `types/index.ts`: добавлена типизация для `problems` в интерфейсе `TextsData`.
- Создан компонент `components/sections/ProblemsSection.tsx`:
  - Реализована сетка `grid-cols-1 md:grid-cols-2`.
  - Номера отформатированы через `padStart(2, '0')` и окрашены в `#F59E0B`.
  - Добавлена анимация `animate-fade-in`.
- Обновлен `components/sections/index.ts` для экспорта компонента.
- Обновлен `app/page.tsx` для отображения секции на главной странице.

## Почему этот подход
- Использование типизированных данных гарантирует отсутствие ошибок при обращении к полям.
- Компонент реализован как Server Component, что соответствует требованиям SSG и производительности.
- Стилизация выполнена исключительно через Tailwind CSS, что минимизирует размер бандла.
