# AI_NOTES — Этап 4: Реализация ProblemsSection

## Что было сделано
- Обновлен интерфейс `TextsData` в `types/index.ts` для поддержки структуры секции проблем.
- Добавлены данные в `lib/data/texts.ts` (заголовок, описание, список из 5 проблем).
- Создан компонент `components/sections/ProblemsSection.tsx` как Server Component.
- Реализована адаптивная сетка (1 колонка на мобильных, 2 на десктопе) с использованием Tailwind CSS.
- Добавлены порядковые номера с акцентным цветом `#F59E0B`.
- Компонент экспортирован через `components/sections/index.ts` и добавлен на главную страницу `app/page.tsx`.

## Почему этот подход
- Использование типизированных данных обеспечивает безопасность типов.
- Server Component подход минимизирует JS-бандл, что соответствует требованиям проекта.
- CSS Grid обеспечивает простоту и производительность без сторонних библиотек.
