# AI_NOTES — Stage 1: Создание библиотеки UI-компонентов

## Что было сделано
- Реализован компонент `Button` с поддержкой вариантов `primary`/`secondary`, состояния загрузки (`isLoading`) и стандартных HTML-атрибутов.
- Реализован компонент `FormInput` (для `input`) и `FormTextarea` (для `textarea`) с поддержкой валидации (`error`) и `forwardRef`.
- Создан `index.ts` для централизованного экспорта компонентов.

## Почему выбран этот подход
- Использованы жестко заданные цвета из палитры проекта (#1B2A4A, #F59E0B) для соответствия требованиям.
- `forwardRef` добавлен для обеспечения совместимости с будущими библиотеками форм (например, React Hook Form).
- Использован `'use client'`, так как компоненты содержат интерактивные состояния (hover, focus, loading).
- Компоненты не имеют внешних отступов (margin), что соответствует NFR-02.
