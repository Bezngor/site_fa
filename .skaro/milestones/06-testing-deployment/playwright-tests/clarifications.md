# Clarifications: playwright-tests

## Question 1
Как именно Playwright должен запускать веб-сервер для тестирования статики из папки out/?

*Context:* В спецификации указано использование 'out/', но не определен инструмент для раздачи статики (serve, npx http-server или встроенный webServer Playwright).

**Options:**
- A) Использовать встроенный в playwright.config.ts 'webServer' с командой 'npx serve out -p 3000'
- B) Запускать тесты без webServer, предполагая, что сервер уже запущен вручную
- C) Использовать 'npx http-server out' в скрипте start:e2e

**Answer:**
Использовать встроенный в playwright.config.ts 'webServer' с командой 'npx serve out -p 3000'

## Question 2
Какое поведение ожидается при проверке видимости секций, если контент подгружается лениво?

*Context:* Next.js и Tailwind могут влиять на скорость отрисовки; простая проверка toBeVisible() может упасть без скролла или ожидания.

**Options:**
- A) Использовать стандартный toBeVisible() (Playwright подождет появления в DOM)
- B) Принудительно скроллить к каждой секции перед проверкой видимости
- C) Проверять только наличие элемента в DOM через toBeAttached()

**Answer:**
Принудительно скроллить к каждой секции перед проверкой видимости

## Question 3
Как обрабатывать проверку ошибок валидации формы (FR-02)?

*Context:* Нужно уточнить, какие именно сообщения об ошибках искать в DOM при пустой отправке.

**Options:**
- A) Искать конкретные тексты ошибок из Zod (например, 'Введите корректный email')
- B) Проверять только наличие атрибутов 'aria-invalid' или специфических классов ошибок
- C) Проверять наличие любого элемента с ролью 'alert' или внутри контейнера ошибок

**Answer:**
Проверять наличие любого элемента с ролью 'alert' или внутри контейнера ошибок

## Question 4

*Context:* Спецификация упоминает якоря, но не уточняет, является ли навигация частью smoke-теста.

**Options:**
- A) Да, добавить тест: клик по ссылке в меню -> проверка URL (hash) -> проверка видимости секции

Нет, достаточно проверить наличие секций по ID на прямой загрузке страницы

## Question 5
