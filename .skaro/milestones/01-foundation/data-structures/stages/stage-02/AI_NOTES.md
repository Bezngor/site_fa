# AI_NOTES — Этап 2: Реализация статических данных

## Что было сделано
- Создан файл `lib/data/texts.ts` с типизированным объектом `texts`, содержащим контент для всех 8 секций лендинга.
- Создан файл `lib/data/cases.json` с массивом из 3 кейсов, соответствующих интерфейсу `Case`.
- Создан файл `lib/data/products.ts` с плоским массивом продуктов, соответствующих интерфейсу `Product`.

## Почему выбран такой подход
- Использование отдельных файлов для данных (`texts.ts`, `products.ts`, `cases.json`) обеспечивает чистоту архитектуры и удобство поддержки.
- Типизация через импорт из `types/index.ts` гарантирует, что структура данных всегда соответствует ожиданиям компонентов.
