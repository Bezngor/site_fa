## Что было сделано
- Создана директория `types/` и файл `types/index.ts` с полными интерфейсами для всех данных (TextsData, Case, Metric, Product, HowWeWorkStep).
- Создан файл `lib/data/texts.ts` с типизированным объектом `texts` для всех 8 секций.
- Создан файл `lib/data/cases.json` с массивом из 3 кейсов, соответствующих схеме `Case`.

- Использование статических файлов соответствует ADR-003 и требованиям SSG.
