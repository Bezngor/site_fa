# Clarifications: project-setup

## Question 1
Какие именно шрифты Google Fonts следует интегрировать в базовую настройку проекта?

*Context:* Конституция упоминает использование Google Fonts через preload, но конкретные гарнитуры не определены в спецификации.

**Options:**
- A) Inter для основного текста и Montserrat для заголовков
- B) Geist (стандартный для Next.js 15) для всего сайта
- C) Roboto для всех элементов как наиболее универсальный вариант
- D) Не использовать внешние шрифты, ограничиться системными

**Answer:**
Inter для основного текста и Montserrat для заголовков

## Question 2
Как должна быть настроена обработка изображений в next.config.js для режима 'export'?

*Context:* Стандартный компонент next/image требует внешнего загрузчика или отключения оптимизации при статическом экспорте.

**Options:**
- A) Использовать unoptimized: true в next.config.js для простоты
- B) Настроить loader: 'custom' для использования стороннего сервиса (например, Cloudinary)
- C) Использовать плагин-заглушку для локальной оптимизации при сборке

**Answer:**
Использовать unoptimized: true в next.config.js для простоты

## Question 3
Какую стратегию именования Tailwind-цветов использовать в конфигурации?

*Context:* Для соблюдения ADR-002 и FR-02 нужно решить, расширять ли стандартную палитру или заменять её.

**Options:**
- A) Добавить кастомные ключи: primary (#1B2A4A), accent (#F59E0B) в секцию extend
- B) Переопределить стандартные цвета (например, blue и amber) на указанные в спецификации
- C) Использовать CSS переменные в globals.css и связать их с tailwind.config.ts

**Answer:**
Добавить кастомные ключи: primary (#1B2A4A), accent (#F59E0B) в секцию extend

## Question 4
Требуется ли настройка базового пути (basePath) в конфигурации Next.js?

*Context:* Если деплой на Vercel идет в корень домена, basePath не нужен, но для GitHub Pages или подпапок он критичен.

**Options:**
- A) Оставить пустым (деплой в корень домена)
- B) Настроить динамическое определение через переменную окружения
- C) Зафиксировать конкретный путь (например, /landing)

**Answer:**
Оставить пустым (деплой в корень домена)

*Context:* Конституция требует соблюдения стандартов кодирования, что проще всего обеспечить жесткими правилами линтера сразу.
