# AI_NOTES — Этап 3: Интеграция шрифтов и базовый layout

## Что было сделано
- Интегрированы шрифты Google Fonts (Inter для основного текста, Montserrat для заголовков) через `next/font/google`.
- Обновлен `app/layout.tsx`: добавлены переменные CSS для шрифтов и применены классы в `html` и `body`.
- Обновлен `app/globals.css`: добавлены стили для заголовков с использованием переменной `font-heading`.
- Обновлен `tailwind.config.ts`: настроены семейства шрифтов `sans` и `heading` для использования переменных CSS.
- Обновлен `app/page.tsx`: добавлен тестовый контент с использованием цветов `primary` и `accent` для проверки корректности стилей.

## Почему выбран этот подход
- Использование `next/font` обеспечивает оптимальную загрузку шрифтов (preload) и отсутствие Layout Shift.
- Переменные CSS позволяют легко управлять шрифтами через Tailwind, сохраняя типизацию и удобство.
