# Спецификация: root-layout

## Контекст
Корневой макет обеспечивает общую структуру приложения, SEO-оптимизацию через Metadata API, подключение шрифтов и глобальных стилей Tailwind CSS (ADR-001, ADR-002).

## User Scenarios
1. **Посетитель:** Видит корректный заголовок страницы и описание в результатах поиска. Загружает страницу с оптимизированными шрифтами без скачков верстки (CLS).
2. **Бот (SEO):** Индексирует метаданные на русском языке и структуру страницы.
3. **Пользователь на любой странице:** Всегда видит навигацию (Header) и подвал (Footer), независимо от конкретного маршрута (включая 404).

## Functional Requirements
- **FR-01: Metadata.** Установить статические метаданные:
    - **Title:** `FactoryAll — Цифровое управление производством с ИИ`
    - **Description:** `Помогаем предприятиям перейти от ручного управления к умным системам планирования и контроля производства. Результат за 3–6 месяцев.`
- **FR-02: Шрифты.** Подключить Google Fonts через `next/font/google` с поддержкой кириллицы:
    - `Inter` — основной текст.
    - `Montserrat` — заголовки.
- **FR-03: Общие компоненты.** Включить компоненты `Header` и `Footer` непосредственно в `layout.tsx`.
- **FR-04: Цветовая схема.** Реализовать только светлую тему (Light Mode) с использованием основной палитры:
    - Primary: `#1B2A4A` (темно-синий)
    - Accent: `#F59E0B` (янтарный)
    - Background: `#FFFFFF` (белый)

## Non-Functional Requirements
- **NFR-01: Производительность.** Preload шрифтов через механизмы Next.js для обеспечения LCP < 2.5s.
- **NFR-02: Адаптивность.** Контейнеры внутри секций должны иметь `max-w-7xl mx-auto`, в то время как сами секции (обертки) поддерживают 100% ширины для фоновых заливок.

## Boundaries (что НЕ включено)
- Логика конкретных секций лендинга (кроме Header/Footer).
- Динамическая смена тем (Dark Mode не поддерживается).

## Acceptance Criteria
- [x] Исходный код страницы содержит указанные Title и Description.
- [x] Шрифты Inter и Montserrat загружаются локально (через next/font) без внешних запросов к Google API в рантайме.
- [x] Header и Footer отображаются на главной странице и на странице 404.
- [x] Отсутствуют ошибки гидратации в консоли браузера.
- [x] Tailwind CSS 4 корректно применяет стили к элементам.
- [x] Фоновые цвета секций растягиваются на всю ширину экрана, а контент ограничен `max-w-7xl`.
