# AI_NOTES — Этап 1: Реализация компонентов Header и Footer

## Что было сделано
- Созданы компоненты `Header` и `Footer` как Server Components.
- `Header` включает навигацию на основе `lib/data/navigation.ts` и кнопку CTA.
- `Footer` включает копирайт и контактный email.
- Реализована адаптивность с использованием `max-w-7xl` и цветовой палитры проекта (`#1B2A4A`, `#F59E0B`).

## Почему выбран этот подход
- Использование Server Components обеспечивает минимальный размер JS-бандла.
- Anchor-ссылки позволяют реализовать навигацию без клиентского состояния, что соответствует требованиям SSG.
- Tailwind CSS обеспечивает быструю стилизацию без лишних CSS-файлов.

## Риски и ограничения
- Навигация полагается на `id` секций, которые должны быть определены в `page.tsx`.
- Отсутствует мобильное меню (бургер), так как в текущем этапе фокус на базовой структуре Server Components.
