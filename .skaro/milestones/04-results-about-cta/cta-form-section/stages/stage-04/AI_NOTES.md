# AI_NOTES — Stage 4: Реализация CtaFormSection

## Что было сделано
- Полностью переработан компонент `CtaFormSection`:
  - Убрал React Context, перешёл на локальный `useState` для управления статусом и ошибками полей (упрощение архитектуры согласно уточнению).
  - Реализовал валидацию только при отправке формы через `contactFormSchema.safeParse`.
  - Интеграция с Formspree: чтение `NEXT_PUBLIC_FORMSPREE_URL` (приоритет) или `NEXT_PUBLIC_FORMSPREE_ID` для построения URL; отправка через `submitContactFormToFormspree` с таймаутом 12 с.
  - При успешной отправке форма заменяется на `SuccessMessage` с текстом из `texts.cta.form.successMessage`.
  - При ошибке/таймауте/отсутствии конфигурации отображается `ErrorMessage` с текстом ошибки и подсказкой (`fallbackHint`), а также кнопка «Отправить через почту», которая открывает почтовый клиент с черновиком письма (mailto:).
  - Очистка полей и сброс состояний при успешной отправке и при начале ввода после ошибки.
  - Добавлены русскоязычные комментарии в коде.

## Почему такой подход
- Упрощение: для одной формы на странице локальный state достаточно, избыточность Context устранена.
- Fallback через кнопку, а не автоматический переход, даёт пользователю контроль и соответствует уточнению.
- Использование `NEXT_PUBLIC_FORMSPREE_URL` как приоритетной переменной окружения позволяет гибко задавать endpoint (например, для тестовых сред), с fallback на ID для обратной совместимости.
- Валидация только на submit соответствует требованиям UX и снижает шум при вводе.

## Риски и ограничения
- Поле honeypot скрыто через `absolute -left-[5000px]` — стандартный подход, но может быть проигнорировано ботами.
- Зависимость от Formspree: при их недоступности форма уходит в fallback, что требует ручной отправки письма.
- Bundle size: добавление состояния и обработчиков незначительно, но должно уложиться в бюджет 150 KB (уже учтено в проекте).
- Кнопка «Отправить через почту» использует `window.location.href`, что может быть заблокировано всплывающими окнами; альтернатива — `mailto` ссылка, но тогда нужно управлять фокусом.
