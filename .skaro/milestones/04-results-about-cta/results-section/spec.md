# Спецификация: results-section

## Контекст
Подведение итогов достижений и демонстрация конкретных бизнес-показателей эффективности решений. Секция служит для укрепления доверия через твердые цифры.

## User Scenarios
1. **Посетитель:** Прокручивает страницу и видит ключевые метрики эффективности, которые подтверждают экспертизу компании и потенциальную выгоду от внедрения.

## Functional Requirements
- **FR-01:** Отображение списка метрик, загружаемых из `lib/data/texts.ts`.
- **FR-02:** Использование конкретных данных:
    - `{ value: '3–6', unit: 'мес', label: 'Срок внедрения MES' }`
    - `{ value: '30%', unit: '+', label: 'Снижение срывов сроков' }`
    - `{ value: '20–35%', unit: '', label: 'Сокращение потерь производства' }`
    - `{ value: '3', unit: 'кейса', label: 'Успешных внедрения' }`
- **FR-03:** Визуальная структура: горизонтальная сетка (Grid) из 4 колонок на десктопе, переходящая в 1-2 колонки на мобильных устройствах.
- **FR-04:** Наличие иконок над числовыми значениями для визуальной идентификации каждой метрики.

## Non-Functional Requirements
- **NFR-01:** Server Component (отсутствие клиентского состояния).
- **NFR-02:** Цветовая схема: Темный фон (`#1B2A4A`), белый текст подписей, акцентный цвет для цифр — Amber (`#F59E0B`).
- **NFR-03:** Отсутствие JavaScript-анимаций (статичный рендеринг для максимальной производительности и SEO).
- **NFR-04:** Адаптивность: корректное отображение на всех типах экранов.

## Boundaries (what is NOT included)
- JS-анимации счетчиков (цифры статичны).
- Интерактивные всплывающие подсказки.
- Ссылки на детальные отчеты внутри карточек метрик.

## Acceptance Criteria
- [ ] Секция отображает все 4 метрики, указанные в FR-02.
- [ ] Фон секции соответствует темно-синему цвету бренда (#1B2A4A).
- [ ] Числовые значения выделены цветом Amber (#F59E0B).
- [ ] Каждая метрика содержит иконку, число с единицей измерения и короткую подпись.
- [ ] Секция является Server Component и не загружает клиентский JS для рендеринга данных.
- [ ] Сетка корректно перестраивается под мобильные разрешения.
