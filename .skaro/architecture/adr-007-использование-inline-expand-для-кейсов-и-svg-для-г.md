# ADR-002: Оптимизация UX и производительности

**Status:** accepted
**Date:** 2023-10-24

## Context
Необходимо обеспечить высокую скорость загрузки и удобство чтения на мобильных устройствах без перегрузки интерфейса.

## Decision
1. **Inline Expand (Accordion)**: Вместо модальных окон для кейсов используем раскрывающиеся блоки. Это сохраняет контекст страницы и упрощает навигацию на мобильных устройствах.
2. **SVG Graphics**: Вместо растровых изображений (PNG/JPG) используем SVG-иконки и CSS-плашки. Это радикально снижает размер страницы и обеспечивает четкость на Retina-дисплеях.
3. **Environment Variables**: ID формы Formspree выносится в `NEXT_PUBLIC_`, чтобы обеспечить гибкость при деплое на разные окружения (staging/prod).

## Alternatives Considered
- Модальные окна (Radix UI/Headless UI): Отклонено для минимизации JS-бандла.
- Библиотеки анимаций (Framer Motion): Отклонено, достаточно CSS transitions для плавности.

## Consequences
- Плюсы: Минимальный вес страницы, мгновенная реакция интерфейса.
- Минусы: Ограниченность визуальных эффектов (только векторная графика).
