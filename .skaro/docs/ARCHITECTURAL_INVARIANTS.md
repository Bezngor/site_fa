# Архитектурные инварианты проекта FactoryAll

Этот документ определяет неизменные правила и ограничения, которым должен соответствовать код проекта. Нарушение этих инвариантов является блокирующим фактором при проверке качества (QA).

## 1. Технологический стек и сборка
- **Фреймворк**: Next.js 15 (App Router).
- **Режим работы**: Исключительно Static Site Generation (SSG). 
- **Конфигурация**: Обязательное наличие `output: 'export'` и `images.unoptimized: true` в `next.config.js`.
- **Стилизация**: Только Tailwind CSS 4. Использование сторонних UI-библиотек (MUI, AntD) или CSS-модулей запрещено.
- **Язык**: TypeScript 5.x со строгой типизацией (`strict: true`).

## 2. Производительность и бюджеты
- **JS Bundle**: Общий размер JavaScript на первой загрузке (gzipped) — не более **150 КБ**.
- **Lighthouse**: Показатели Performance и SEO должны быть выше **90**.
- **Изображения**: Только формат **WebP**. Все изображения должны быть оптимизированы и находиться в `public/images/`.
- **Шрифты**: Локальное подключение через `next/font/google` (Inter и Montserrat).

## 3. Стандарты кодирования
- **Компоненты**: Server Components по умолчанию. Директива `'use client'` допустима только для интерактивных элементов (формы, аккордеоны).
- **Размер функций**: Максимальная длина любого компонента или функции — **40 строк**.
- **Вложенность**: Максимальная глубина вложенности логических блоков в одном файле — **3 уровня**.
- **Локализация**: Весь контент, комментарии и документация — только на **русском языке**.

## 4. Данные и состояние
- **Хранение**: Только статические TS/JSON файлы в `/lib/data/`. Использование внешних баз данных или API в рантайме запрещено.
- **Формы**: Валидация через **Zod**. Отправка данных только через Formspree.
- **Состояние**: React Context разрешен только для локального управления состоянием форм или модальных окон.

## 5. Дизайн и палитра
- **Цвета**: 
  - Основной (Primary): `#1B2A4A` (темно-синий).
  - Акцентный (Accent): `#F59E0B` (янтарный).
  - Фон: `#FFFFFF` (белый) или `#F8F9FA` (светло-серый).
- **Адаптивность**: Mobile-first подход, обязательная поддержка разрешений от 320px до 1920px.

## 6. Безопасность и SEO
- **Метаданные**: Обязательное заполнение Title, Description и OpenGraph в `layout.tsx`.
- **Заголовки**: Настройка CSP и заголовков безопасности в `vercel.json`.
- **Спам**: Наличие Honeypot-поля во всех формах обратной связи.
